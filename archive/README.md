# WA-HTML Viewer

Static web viewer for chat exports generated by **WhatsApp HTML Builder**.  
Works on GitHub Pages, Netlify, Vercel — no backend required.

## Setup

### 1. Add your chats

Copy the output folder from WhatsApp HTML Builder into `/chats/`:

```
viewer/
├── index.html
├── style.css
├── script.js
└── chats/
    ├── index.json          ← manifest (required)
    ├── my-chat/
    │   ├── chat.html
    │   ├── config.json
    │   └── media/
    └── another-chat/
        ├── chat.html
        ├── config.json
        └── media/
```

### 2. Create chats/index.json

This file tells the viewer which folders to load (needed for static hosting):

```json
["my-chat", "another-chat"]
```

Or as objects with display names:

```json
[
  { "name": "my-chat" },
  { "name": "another-chat" }
]
```

Alternatively, use `chats/manifest.txt` (one folder per line):

```
my-chat
another-chat
# lines starting with # are ignored
```

### 3. Deploy

**GitHub Pages:**
```bash
git init
git add .
git commit -m "deploy viewer"
git remote add origin https://github.com/username/repo
git push -u origin main
# Enable GitHub Pages in repo settings → branch: main
```

**Netlify / Vercel:**  
Drag & drop the `viewer/` folder into the dashboard, or connect your repo.

**Local testing:**
```bash
npx serve .
# or
python3 -m http.server 8080
```

> ⚠️ Do NOT open `index.html` directly with `file://` — fetch() requires an HTTP server.

## Features

- 📱 Responsive: sidebar + chat on desktop, full-screen on mobile
- 👤 "View as" toggle: switch which side is "you" — saved to localStorage
- 🖼️ Images (click to expand), 🎥 Video, 🔊 Audio, 📄 Documents, 🔰 Stickers
- 🔗 Google Drive embeds via `<iframe>`
- 🔍 Sidebar search filter
- ⚡ No page reloads, pure DOM/fetch

## config.json fields

| Field | Type | Description |
|-------|------|-------------|
| `me` | string | Default "you" — determines right-side bubbles |
| `participants` | string[] | All chat participants |
| `chatName` | string | Display name |
| `totalMessages` | number | Optional: shown in sidebar |

## File Structure

```
index.html   — app shell, layout, settings panel
style.css    — all styles (CSS variables, responsive)
script.js    — all logic (chat loading, rendering, view-as)
chats/       — your chat exports go here
```
